NAME
    whoami - print effective user ID

DESCRIPTION
    The whoami command prints the effective user ID of the current user.

SYNOPSIS
    whoami

OPTIONS
   --help      		Display Help
   --version   		Display program version info

EXAMPLE
    whoami

NAME
    wait - wait for a process to complete

DESCRIPTION
    The wait command waits for a specified process to complete.

SYNOPSIS
    wait [n ...]

OPTIONS
    n   A process ID or a job specification

EXAMPLE
    wait 1234
    wait

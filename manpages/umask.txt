NAME
    umask - set file mode creation mask

DESCRIPTION
    The umask command sets the file mode creation mask.

SYNOPSIS
    umask [OPTION]... [MASK]

OPTIONS
   mode  	File creation mask
   -S    	Print the mask in symbolic format
   -p    	Output in a form that can be reused as input

EXAMPLE
    umask
    umask 022

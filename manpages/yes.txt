NAME
    yes - output a string repeatedly until killed

DESCRIPTION
    The yes command outputs a string repeatedly until killed.

SYNOPSIS
    yes [STRING]

OPTIONS
   text       	The text to repeat
   --help     	Display help
   --version  	Display the version no.

EXAMPLE
    yes
    yes "y"

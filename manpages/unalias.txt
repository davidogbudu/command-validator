NAME
    unalias - remove alias definitions

DESCRIPTION
    The unalias command removes alias definitions.

SYNOPSIS
    unalias [OPTION]... [ALIASES]...

OPTIONS
    -a, --all                 remove all aliases

EXAMPLE
    unalias ls
    unalias -a
